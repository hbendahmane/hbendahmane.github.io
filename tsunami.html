Tsunami Propagation Simulation

Objective

To simulate the propagation of a tsunami wave triggered by a seismic event in a realistic oceanic region, compute the arrival time at nearby coasts, the wave height, front velocity, and impact force. The goal is to demonstrate a high-level engineering approach, integrating bathymetric data, fluid mechanics, and geophysical parameters.

Parameters & Data Sources

  • Epicenter location: Banda Sea, Indonesia (USGS)
  • Seismic magnitude: 7.8 Mw
  • Ocean depth / bathymetry: GEBCO data
  • Wave speed: computed using shallow-water approximation √(g·h) (g=9.81 m/s², h=depth in meters)
  • Wave height attenuation: based on empirical coastal models (source: Coastal Engineering, 2005)

Code Highlights

The simulation was implemented in C for high performance and precision in wave propagation calculations. The code computes wavefront positions, arrival times, and impact forces using bathymetric and seismic data. 


// Example C snippet: calculate wave speed at given depth
double g = 9.81; // gravity
double depth = 4500; // ocean depth in meters
double wave_speed = sqrt(g * depth); // shallow-water approximation

// Compute time to reach coast
double distance_to_coast = 150000; // meters
double arrival_time = distance_to_coast / wave_speed;
printf("Wave arrives in %.2f seconds\n", arrival_time);
